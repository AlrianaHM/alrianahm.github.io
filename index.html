  • Comic Book API

    An API built with ASP.NET Web API framework. There are Create, Read, Update, and Delete (CRUD) API for Comic and Book data. There also Search Endpoint to searh the comic.
    Entity framework is used to connect model and database in the sql server. To encapsulate the data structure, data transfer object (DTO) is used as the return value of the API. Each API request is logged using log4net and saved to the file.

    Built Using:

    • ASP.NET
    • C#

  • Counting Oil Palm Bunch with Image Analysis

    Counting how many oil palm bunch from the given image. There is three step, image enhancement, imahe segmentation, and image counting. In the image enhancement the methods used are Graylevel Transformation, Histogram Equalization, and Median Filtering. For image segmentation, image binarization, dilation and erosion prosess is used. The last step is counting using connected component labelling and clustering.

    • Java

  • Web Calculator

    A calculator that can be accessed in the browser. The UI is created using ASP.Net webform component. Each button will create a math expression. Pressing the "Equal" Button will compute the expression using datatable compute method. If the expression is not valid it will return "syntax error". There also additional button like clear and delete button.
